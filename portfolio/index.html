Artist Statement

Scientific ideas can be tender.

This version merges the crystalline visual language of the original landing page with the richer interaction logic of the simulation gallery, reshaping them into a more cohesive portfolio experience for graduate applications and long-term presentation.

I approach code as both structure and light: a medium for turning abstraction into sensation. Orbital systems, chaos attractors, interference patterns, and fractals are not only scientific objects to me — they are visual metaphors for intimacy, memory, recursion, and scale. I want my websites to feel like inhabitable worlds: precise enough to be legible, yet soft enough to feel almost impossible.

Practice

What this portfolio is trying to prove.

Visual language

The site preserves the original “star-crystal” mood — aurora haze, glass surfaces, nacre highlights, drifting particles — but now uses that language more consistently across sections, previews, interactions, and typography.

Technical framing

The gallery architecture has been reimagined as a portfolio-ready interface rather than a loose experiment. It now reads as curation: a display system with categories, navigation, focus states, and multiple scientific domains under one coherent shell.
